# Face-map illustration styles

Reference art for the (prototype) interactive face-map feature — a single-line
woman's face where customers hover/tap regions (forehead, eyelids, lips) to see
available aesthetic procedures and prices.

- **`face-a-soft-beauty.svg`** — **CHOSEN.** Refined feminine features (brows,
  lashes, subtle nose, full lips), framing hair, gold pupils.
- **`face-c-minimal-glow.svg`** — backup direction kept for discussion with the
  clinic owners. Ultra-simple: hair, almond eyes, lips, no nose.

Both use white stroke on the green/gold brand and share `viewBox="0 0 200 250"`.
Decided 2026-06-04 during brainstorming. Feature is exploratory and must remain
easy to roll back.
